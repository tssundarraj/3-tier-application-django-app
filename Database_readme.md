## üîå How to Connect (General Steps)

1. **Install the respective DB** server.
2. **Open ports** in firewall/security group if needed.
3. Use the **connection tool** with:
   - **Host/IP** of the DB
   - **Port**
   - **Username/Password**
   - **DB name** (if required)
